Kotlin 委托（by lazy）

发表于2022-08-03|更新于2022-08-03|kotlin 笔记
|阅读量:
// initializer 为传进来的 lamda 表达式
private class SynchronizedLazyImpl<out T>(initializer: () -> T, lock: Any? = null) : Lazy<T>, Serializable {
    private var initializer: (() -> T)? = initializer
    @Volatile private var _value: Any? = UNINITIALIZED_VALUE
    // SynchronizedLazyImpl 对象作为 lock
    private val lock = lock ?: this
    override val value: T
        get() {
            val _v1 = _value
            // 初始化过就直接返回
            if (_v1 !== UNINITIALIZED_VALUE) {
                @Suppress("UNCHECKED_CAST")
                return _v1 as T
            }
            // 否则同步⽅式通过传进来的 initializer 创建新的对象
            return synchronized(lock) {
                val _v2 = _value
                if (_v2 !== UNINITIALIZED_VALUE) {
                    @Suppress("UNCHECKED_CAST") (_v2 as T)
                } else {
                    // 传进来的 initializer 不能创建 null 对象 否则抛出异常
                    val typedValue = initializer!!()
                    _value = typedValue
                    initializer = null
                    typedValue
                }
            }
        }
    override fun isInitialized(): Boolean = _value !== UNINITIALIZED_VALUE
    override fun toString(): String = if (isInitialized()) value.toString() else "Lazy value not initialized yet."
    private fun writeReplace(): Any = InitializedLazyImpl(value)
}
```s

- 线程安全
- 传进来的 lamda initializer 不能创建空对象
- 因为 SynchronizedLazyImpl 没有 setValue() ⽅法，所以 lazy 只⽤于常量 val。
