数组—盛水最多的容器

发表于2022-08-03|更新于2022-08-03|小镇做题家
|阅读量:

链接：https://leetcode-cn.com/problems/container-with-most-water

给定一个长度为 n 的整数数组 height 。有 n 条垂线，第 i 条线的两个端点是 (i, 0) 和 (i, height[i]) 。

找出其中的两条线，使得它们与 x 轴共同构成的容器可以容纳最多的水。

返回容器可以储存的最大水量。

第一版：

思路：枚举计算所有可能，选出最大面积（在leetcode中超时🤣）

第二版：

思路：利用双指针，左右边界选定两端，则宽度为最大，然后两个比较两个指针所在位置的高度，低的一方则往中间收敛，并在过程中计算面积

代码优化：

利用三目运算简化 if else 并执行 对指针的操作，但是 i 和 j 的值有变化导致差值会少 1 ，计算面积时要加 1

利用 while 实现双指针(相比for循环更加易读）：
